题目描述

！！！内存限制4MB！！！

经过一段时间的紧张筹备，电脑小组的“RP餐厅”终于开业了，这天，经理LXC接到了一个定餐大单，可把大家乐坏了！员工们齐心协力按要求准备好了套餐正准备派送时，突然碰到一个棘手的问题，筷子！CX小朋友找出了餐厅中所有的筷子，但遗憾的是这些筷子长短不一，而我们都知道筷子需要长度一样的才能组成一双，更麻烦的是CX找出来的这些筷子数量为奇数，但是巧合的是，这些筷子中只有一只筷子是落单的，其余都成双，善良的你，可以帮CX找出这只落单的筷子的长度吗？

输入格式

第一行读入一个数N，它代表CX找到的筷子的根数。

第二行是N个用空格隔开的数，代表筷子的长度。

输出格式

一行，落单的筷子的长度。

题解

Subtask #1 (60 pts)

开一个数组，对其进行排序，我们暴力扫一遍，判断哪个数出现了奇数次，哪个数就是答案。

#include <iostream>
#include <algorithm>
using namespace std;

int n,a[1000005];

int main()
{
    ios::sync_with_stdio(false);
    cin>>n;
    for(register int i=1;i<=n;i++)
        cin>>a[i];        
    sort(a+1,a+n+1);
    for(register int i=1;i<=n;i++)
    {
        int cnt=1;
        for(register int j=i+1;a[j]==a[j-1];j++)
        {
            cnt++;
            i=j;;
        }
        if(cnt % 2 != 0)cout<<a[i];
    }
}

正解：位运算

不知大家有没有听过异或运算。

异或运算的性质：

  • 如果两个数为1，就为0
  • 如果两个数为0，就为0
  • 如果两个数有一个为1，另一个为0，就为1

其实就是二进制下的不进位加法。

我们还能发现：

  1. 两个相同的数进行异或运算后一定为0
  2. 任何数异或0都不变

所以，我们只需对所有的数异或一遍，最后剩的数即为答案。

代码

#include <iostream>
#include <algorithm>
using namespace std;
int n, a, ans;
int main()
{
    cin>>n;
    for(register int i=1;i<=n;i++)
    {
        cin >> a;
        ans ^= a;
    }
    cout << ans << endl;
}
